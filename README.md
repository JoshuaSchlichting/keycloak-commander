# keycloak-commander

## Use
### Create config file
```
Usage:
  keycloak-commander init [flags]

Flags:
  -h, --help              help for init
  -i, --host string       Hostname of the Keycloak server
  -p, --password string   Admin's password
  -r, --realm string      Realm to use
  -u, --username string   Admin's username
```
